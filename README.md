# NIFTY-50_analysis

Nifty 50 Market Trend Analysis

Overview: A Python-based analysis tool that fetches real-time financial data for the Nifty 50 index and visualizes key technical indicators to identify market trends.

Tech Stack: Python, Pandas, Matplotlib, yFinance.

Key Features:

Automated data extraction from Yahoo Finance.

Calculation of 50-day and 200-day Simple Moving Averages (SMA).

Implementation of RSI (Relative Strength Index) logic for momentum analysis.
